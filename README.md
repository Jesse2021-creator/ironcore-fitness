# 🏋️ IronCore Fitness Website

A modern, responsive fitness landing page built with **HTML, CSS (Tailwind), and JavaScript**.  
Designed for performance, smooth animations, and a premium gym/fitness brand feel.

## 🧠 Features

- ⚡ Smooth scroll animations
- 🌀 GSAP-based section animations
- 🎯 Scroll progress bar
- 🧭 Smart navbar hide/show on scroll
- 🎥 Hero image parallax effect
- 💬 Auto sliding testimonial carousel
- ⌨️ Keyboard navigation (left/right arrows)
- 📱 Fully responsive design
- 🎨 Modern glassmorphism UI
- ✨ Intersection Observer reveal animations

---

## 🛠️ Built With

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- GSAP (ScrollTrigger)
- Google Fonts
